# 🛠️ First Installation - ifcSQL Tools for QGIS

Here you will find the PDF documentation and executable scripts (`.bat`) needed to install and set up the **ifcSQL Tools for QGIS** plugin.

## 📁 Folder Contents

The files are numbered to guide you step-by-step through the setup process:

* 📄 **`0.Start with ifcSQL_Tools.pdf`**
    **Starting point.** This guide will walk you through the necessary steps to install and configure all required components.
    
* ⚙️ **`1.ifcSQL_Installation_MSSQL.bat`**
    Batch script to automate the database installation and configuration in **Microsoft SQL Server (MSSQL)**.
    
* 📄 **`2.Create a user in MSSQL.pdf`**
    A detailed guide explaining how to create and properly set permissions for a new user within MSSQL.
    
* ⚙️ **`3.ifcSQL_Installation_PostgreSQL.bat`**
    Batch script to automate the database installation and configuration in **PostgreSQL**.

## 🚀 How to Use

**Start reading:** Open the `0.Start with ifcSQL_Tools.pdf` file and follow the instructions. As you follow the instructions, you will be asked to open the other files as well.
